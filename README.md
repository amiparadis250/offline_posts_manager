# Offline Posts Manager

Flutter app for managing posts locally using SQLite, no internet needed.
## Features
- View, add, edit, delete posts
- Pick images from gallery (stored as base64 in SQLite)
- Search posts
- Fully offline

## Dependencies
- **sqflite** 
- **path** 
- **image_picker**
